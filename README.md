# Store
redux studies Store

***A store holds the whole state tree of your application.***

## Clone the repo & install
```
git clone https://github.com/suxxuscomp/store.git
cd ./create-store
npm install
```
## Usage
cd ./src/scripts

**node index.js**

## Tips
* Is an object with a few methods on it.
* getState()
* dispatch(action)
* subscribe(listener)
* replaceReducer(nextReducer)
